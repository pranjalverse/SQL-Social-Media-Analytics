# SQL-Social-Media-Analytics
This project focuses on analyzing social media data using SQL to extract meaningful insights about user activity, engagement, and platform growth. The database is designed with multiple relational tables including users, posts, comments, likes, and follows, enabling comprehensive analysis of interactions across the platform .

The project involves writing optimized SQL queries to solve real-world analytical problems such as identifying active users, measuring engagement, and tracking follower growth over time. It demonstrates strong understanding of joins, aggregations, subqueries, and data relationships.

Key analysis includes calculating total posts per user, average likes per post, identifying the most active users, and finding high-engagement posts based on likes and comments. Additional insights include detecting inactive users, analyzing comment activity, tracking monthly follower growth, and identifying mutual connections between users.

Overall, this project showcases the ability to design relational databases, perform advanced SQL queries, and derive actionable insights from structured data, making it highly relevant for data analyst and data science roles.
